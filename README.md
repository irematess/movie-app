## Movie App

## Project Setup

```sh
git clone [https://github.com/irematess/e-commerce-app.git](https://github.com/irematess/movie-app.git)
```

```sh
cd movie-app
```

```sh
yarn / npm install
```

### Compile and Hot-Reload for Development

```sh
yarn dev / npm run dev
```
